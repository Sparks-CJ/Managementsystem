 SDG 12: Food Management System

A MERN Stack Application promoting Responsible Consumption & Production

Overview

This project is built as part of the MERN Stack Specialization and aligns with UN Sustainable Development Goal 12 (Responsible Consumption and Production).

The Food Management System helps reduce food waste by enabling users to:

Track food inventory

Monitor expiry dates

Donate unused food

Receive notifications before items expire

Promote sustainable food practices
